MARGOT

Margot se creo con la intension de que el dieño web fuera mas limpio y sencillo que cualquiera que decidiera arrancar una aventura como frontend entendiera cada linea que escribiera y no utilizara frameworks llenos de herramientas que no se sabian para que servian, este marco de trabajo es muy sencillo y por esto es muy customisable ademas de tener todo el poder para ser escalable.

Empezando

  2. Modifica el package.json a tu favor
  3. Empieza a codear
